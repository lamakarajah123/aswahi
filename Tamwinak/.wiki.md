# Project Summary
**Tamweenak** is a Progressive Web App (PWA) designed for grocery ordering and delivery, facilitating seamless connections between customers, grocery stores, delivery drivers, and administrators. Customers can conveniently order groceries from approved stores within a 15 km radius. Delivery drivers receive nearby orders to fulfill, while the admin panel oversees operations, ensuring a smooth experience for all users. The project has recently integrated an email/password authentication system, replacing the previous SSO method, and enhanced security through a comprehensive Role-Based Access Control (RBAC) system.

# Project Module Description
- **Frontend**: A mobile-first, responsive PWA that includes interfaces for customers, stores, drivers, and administrators. The frontend now features an email/password login system, replacing the SSO method, and includes role management and permission settings.
- **Backend**: A REST API with secure authentication, role-based access, and a scalable architecture. Recent updates include a new email/password login endpoint and a seeding endpoint for test accounts.
- **Database**: PostgreSQL for efficient relational data management, optimized for geo-queries and enhanced with new tables for user passwords and roles.

# File Description Inventory
- **backend/**: Contains all backend-related files, including API routes, database models, and service logic. Recent additions include email/password authentication and a seeding mechanism for test accounts.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS, Vite
- **Backend**: FastAPI, SQLAlchemy, PostgreSQL, JWT for authentication, bcrypt for password hashing
- **Deployment**: Docker for containerization

# Usage
1. **Install Dependencies**:
   - For backend: Navigate to `app/backend` and run `pip install -r requirements.txt`.
   - For frontend: Navigate to `app/frontend` and run `pnpm install`.
   
2. **Build**:
   - For frontend: Run `pnpm run build` in the `app/frontend` directory.

3. **Run**:
   - Start the backend server by executing `uvicorn main:app --reload` in the `app/backend` directory.
   - Serve the frontend using a local server setup.
